Chicken farm management web app.

Manage:
1. Chickenhouses
2. Chickens
3. Register of laid eggs
4. Worker's duties
5. Supplies and transactions in the meatshop.

I reapeat, it's not a game, it's a management app with funny cartoonish graphics!
